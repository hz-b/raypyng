<https://github.com/hz-b/raypyng/tree/main/examples>`_ and are ordered from
low-level building blocks to more complete beamline workflows.

- `Flux <https://github.com/hz-b/raypyng/tree/main/examples/10_recipee_Flux>`_

- `Roughness and slopes <https://github.com/hz-b/raypyng/tree/main/examples/12_recipee_slopes_roughness>`_
